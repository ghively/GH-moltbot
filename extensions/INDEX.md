# Extensions

This directory contains extensions (addons) for Moltbot, primarily channel integrations.

## Contents

*   **`whatsapp/`**: WhatsApp channel plugin.
*   **`discord/`**: Discord channel plugin.
*   **`slack/`**: Slack channel plugin.
*   and others...

For development guide, see [ADDON_DEVELOPMENT.md](../../ADDON_DEVELOPMENT.md).
