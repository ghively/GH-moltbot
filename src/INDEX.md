# Source Code

This directory contains the core source code for Moltbot.

## Key Directories

*   **`gateway/`**: Gateway system implementation.
*   **`agents/`**: Agent runtime and tools.
*   **`channels/`**: Channel integrations shared logic.
*   **`config/`**: Configuration management.
*   **`cli/`**: CLI entry points and logic.
*   **`utils/`**: Shared utilities.

For detailed architecture, see [ARCHITECTURE.md](../../ARCHITECTURE.md).
